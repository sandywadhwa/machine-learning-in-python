  Using pandas
----------------
1. Create virtualenv 
    virtualenv -p python3 env
2. Source it
    source env/bin/activate
3. Install pandas
    pip install pandas
4. Refer codes/pandas/pandas-1...py
5. Run code
    python pandas-1....py
